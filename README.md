# Face Recognition System using Python & OpenCV

This project implements a face recognition system using Python and OpenCV with Haar Cascade classifier.

## Features
- Real-time face detection using webcam
- Face recognition from image dataset
- Image preprocessing utilities

## Project Structure
- camera_test.py : Webcam testing
- face_recognition_app.py : Main face recognition logic
- fix_images.py : Image preprocessing
- haarcascade_frontalface_default.xml : Haar Cascade model
- dataset/ : Training images

## Technologies Used
- Python
- OpenCV
- NumPy
